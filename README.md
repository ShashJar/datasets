# datasets

Source: UCI Machine Learning Repository [https://archive.ics.uci.edu/ml/datasets/human+activity+recognition+using+smartphones].

Source: UFHealth Biostatistics Open Learning Textbook [http://bolt.mph.ufl.edu/2012/08/02/learn-by-doing-exploring-a-dataset/]. The dataset has been artificially biased for the purposes of teaching propensity score matching.
